Tecnologias

Html 5

É utilizada como marcação para desenvolver páginas e documentos eletrônicos para a internet. Isso significa que ela garante a formatação ideal para sites.

CSS 3

A função do CSS é justamente otimizar o aspecto visual das páginas, de uma maneira mais organizada e menos complexa na hora de realizar a manutenção.

Javascript

O JavaScript é usado para adicionar interatividade a uma página web e criar aplicações web ricas.
